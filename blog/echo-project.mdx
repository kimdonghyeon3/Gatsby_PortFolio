<h3> 📌 프로젝트 요약 </h3>
<div>
    &nbsp;DK GUIDE 는 학교지리에 익숙치 않은 사용자를 위한 네비게이션 기반 앱으로써
    증강현실(AR) 기술을 활용하여 사용자가 평면 지도를 활용한 길찾기 방식보다 보다 쉽게 길을 찾을 수 있도록 도와주는 가이드앱이다.
    <br/><br/>
    &nbsp;또한 사용자는 해당 목적지에 도착시 해당 위치나 건물에 대한 짧은 정보를 AR 애니메이션 캐릭터를 통해 음성으로 얻을 수 있다.
</div>
<hr/>
<h3> 📌 기술 스택 </h3>
<div>
JAVA, Android, Unity, Mapbox
</div>
<hr/>
<h3> 📌 ‍역할 </h3>
<div>
    <strong>&nbsp;# AR</strong>
    <br/><br/>
    <strong>&nbsp;&nbsp;Navigation 기능</strong><br/> &nbsp;&nbsp;&nbsp;Mapbox, AR CORE, AR foundation을 활용하여 3D선으로 목적지를 안내하는 네이게이션 기능 구현
    <br/><br/>
    <strong>&nbsp;&nbsp;이미지 트레킹 기능</strong><br/> &nbsp;&nbsp;&nbsp;목적지 도착시 특정 이미지를 스캔하여 건물을 안내하는 문구+음성안내 기능 구현
    <br/><br/><br/>
    <strong>&nbsp;# Android</strong>
    <br/><br/>
    <strong>&nbsp;&nbsp;다국어 지원</strong><br/> &nbsp;&nbsp;&nbsp;설정 파일을 활용한 한국어, 중국어, 영어 3개 국어 지원 기능 구현
    <br/><br/>
    <strong>&nbsp;&nbsp;Unity 병합</strong><br/> &nbsp;&nbsp;&nbsp;만들어진 Unity 기능과 Android 기능을 하나의 파일로 병합
    <br/>
</div>
<hr/>
<h3> 📌 후기 </h3>
<div>
    &nbsp;처음 접하는 AR, Android 분야다 보니, 많은 시행착오가 존재했다. 특히, Mapbox와 호환이 되는 Unity 버전을
    찾아가는 과정에 많은 시간을 쏟아야 했다. Unity의 버전 업그레이드는 이전의 기능을 담고 있기는 했지만,
    Mapbox관련 사항은 제거된 기능도 많았기에, 문서를 일일이 찾아보고, 직접 Mapbox와 Unity에 문의하여
    호환되는 버전을 찾아보면서, 프로젝트의 버전의 관리의 중요성을 느껴볼 수 있었다.
    또한, 익숙치 않은 협업으로 코드를 병합하거나 관리함에 있어서 반듯이 규정화된 틀이 필요함을 알 수 있었다.
    <br/><br/>
    &nbsp;이번 프로젝트를 통해서 새로운 기술 스택을 많이 접해볼 수 있었고, 버전관리 / 협업에 관해서 곰심해 볼 수 있는 계기가 되었다.

</div>
