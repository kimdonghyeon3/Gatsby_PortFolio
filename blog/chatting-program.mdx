<h3> #프로젝트 요약 </h3>
<div>
    Java 기초 실습을 활용하기 위해 시작한 프로젝트로 서버 - 클라이언트 구조의 채팅 프로그램을 구현하였다.
    서버는 멀티 쓰레드 방식으로 구현하여, 클라이언트 입장시 하나의 스레드가 담당한다.
    해당 프로그램에서는 채팅방을 만들어 채팅할 수 있으며, 쪽지 보내기, 퇴장의 기능이 구현되었다.
</div>

<h3> #기술 스택 </h3>
<div>
    Java, Java Swing, Socket
</div>

<h3> #역할 </h3>
<div>
    <strong>* Server</strong>
    <br/><br/>
    <strong>MultiThread</strong> : 사용자의 요청에 따른, MultiThread 처리 로직 구현
    <br/><br/>
    <strong>쪽지 기능</strong> : 채팅 이외의 쪽지를 보낼 수 있는 기능 구현
    <br/><br/>
    <strong>채팅 방 기능</strong> : 카톡 방 처럼 채팅을 할 수 있는 방 기능 구현
    <br/><br/>
    <strong>채팅 기능 구현</strong> : Socket을 활용한 실시간 채팅 기능 구현
    <br/><br/><br/>
</div>

<h3> #후기 </h3>
<div>
    Java를 학습해 보면서 처음 시도해본 프로젝트로, Server Client 구조의 서비스에 대해 이해해 볼 수 있었다.
    또한, Java에서 제공하는 Thread 기능을 활용하여, 사용자 별로 하나의 Thread 할당으로 병렬적으로
    로직을 처리할 수 있도록 구현해 보면서, 프로세스와 쓰레드의 관계를 명확히 정의할 수 있었다.

    <br/><br/>
    이번 프로젝트를 통해서 Java의 기본 로직을 서비스에 적용해 볼 수 있었고, 더 나아가 서비/클라이언트의 구조와
    프로세스와 쓰레드의 관계와 같이 많이 사용되는 it의 기본 지식을 쌓아보는 경험이 되었다.

</div>
