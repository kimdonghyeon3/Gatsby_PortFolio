<h3> 📌 후기 </h3>
<div>
    &nbsp;TDD 개발 방법론을 직접 적용해보고, JSON을 직접 Parsing 해보면, DB와 같이 영속성을 유지하기 위해 Local file을 활용해보았다.
    TDD의 개발이 초반에 시간적으로 늦어질 수 있어도 개발을 진행하다보면, 유지 / 보수의 과정에서 테스트의 과정을 쉽게 점검하여
    결과론 적으로 더 빠른 개발을 이어갈 수 있도록 만들어줌을 배워볼 수 있었다.
    <br/><br/>
    &nbsp;또한, 라이브러리로 사용하는 JSON Parsing이나 Local file을 활용해 보면서, JSON 형태의 데이터를 다루는 방법을 익혀볼 수 있었고
    DB가 왜 생겨났고 Local file의 한계점등을 이해해볼 수 있는 프로젝트 였다.
