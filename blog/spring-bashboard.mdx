<h3> 📌 프로젝트 요약 </h3>
<div>
    &nbsp;Spring boot를 연습해보면서, 기본적인 게판의 CRUD를 익혀보며,
    더 나아가 JPA/Paging/Security/Mysql/Search을 활용해 보면서 실제 운영하는 사이트의 구조를 이해하고 재현해 본 프로젝트이다.

</div>
<hr/>
<h3> 📌 기술 스택 </h3>
<div>
    JAVA, Spring, MySQL, JPA, Security
</div>
<hr/>
<h3> 📌 역할 </h3>
<div>
    &nbsp; 1. 기본 게시글 CRUD 기능 구현
    <br/>
    &nbsp; 2. 게시글 리스트 Paging 기능 구현
    <br/>
    &nbsp; 3. 게시판 검색 기능 구현
    <br/>
    &nbsp; 4. security 기반 로그인, 로그아웃 기능 구현
    <br/>
    &nbsp; 5. 추천, 댓글 기능 구현
</div>
<hr/>
<h3> 📌 후기 </h3>
<div>
    &nbsp; 새로운 도전을 하기 전에 기존의 기본적인 페이지를 이해해야 한다. 이러한 사고를 통해서 본 프로젝트를 진행해 보게 되었다.
    <br/><br/>
    &nbsp;Spring에서 자주 사용되는 라이브러리를 활용해 보고, 필수적이지만 많이 접해보지 못하는 검색, 페이징 기법을 직접 프로젝트에 적용해 보면서
    프레임워크의 활용 능력을 키워볼 수 있었든 프로젝트이다.
    <br/><br/>
    &nbsp;또한, JPA와 MySQL을 활용한 DB 연동 및 활용 과정을 객체지향적으로 관리해 보면서 DB의 관계 설정에 대해서도 객체지향적으로 사고해볼 수 있었다.
</div>
